方凯瑞的案例有两个层次：年轻时她作为模特不符合市场审美，于是转去做经纪；行业后来受 AI 和自媒体冲击，她又选择自己下场做内容，先把新模式跑通，再判断能否带员工和模特一起转。

## 二、核心决策场景拆解

### 场景 1：不被模特市场选择时，要不要换到经纪位置

> **一句话决策：** 她发现自己的脸不被品牌选择后，没有继续困在模特身份里，而是转去做模特经纪。
> **核心变量：** 身高和身材符合条件，但长相不符合当时市场审美；身边有模特朋友和市场需求。
> **主要代价：** 需要放弃被选择的期待，转为主动找人、找客户、做交易。

#### 1. 当时约束

- 她说自己做模特时“没有一个品牌会用”。
- 早期资源有限，甚至去酒吧找模特、请朋友拍第一张模卡。

#### 2. 备选项

| 选项 | 内容 |
| --- | --- |
| A | 继续等待品牌选择自己。 |
| B | 离开模特行业。 |
| C | 转做经纪，把身边模特和市场连接起来。 |

#### 3. 最终选择

做模特经纪，并逐步创办/经营 Easy 相关业务。

#### 4. 行动路径

- 在市场里寻找外模机会。
- 用低成本方式帮模特拍卡、接工作。
- 与合伙人合作，逐步从外模扩展到中模和更多业务。

#### 5. 结果

| 阶段 | 结果 |
| --- | --- |
| 短期 | 从被动等待工作变成主动组织资源。 |
| 中长期 | 公司最多有多地分支、数百员工和大量模特流动。 |

#### 6. 代价

需要处理模特、客户、政府/文旅等多方关系，管理复杂度高。

#### 7. 关键变量

- 从被选择转为做连接者。
- 市场机会判断。
- 合伙人互补和信任。

#### 8. 可参考人群

在原岗位不被市场选择、但仍想留在行业里的人。

### 场景 2：管理年轻模特和员工时，要不要只靠控制

> **一句话决策：** 她选择高容错和合作式管理，但在客户越界时明确保护模特。
> **核心变量：** 模特平均年龄很轻、不听话是常态；员工要的是发展前途和钱够不够；客户也可能越界。
> **主要代价：** 老板要承担大量突发问题和夜间救火。

#### 1. 当时约束

- 公司像“托儿所”，年轻模特可能夜店、迟到、闯祸。
- 客户可能提出模特不愿意接受的拍摄要求。

#### 2. 备选项

| 选项 | 内容 |
| --- | --- |
| A | 高压控制所有人。 |
| B | 完全放任。 |
| C | 允许犯错，但在底线问题上立即介入。 |

#### 3. 最终选择

允许员工和模特犯错，同时建立保护和合作边界。

#### 4. 行动路径

- 把员工关系理解为长期合作，而非离职即翻脸。
- 当客户要求模特做未提前谈清的情趣化拍摄时，她选择拉人走。

#### 5. 结果

| 阶段 | 结果 |
| --- | --- |
| 短期 | 组织能容纳年轻人的不稳定。 |
| 中长期 | 她把管理关系从服从转成发展、合作和互相成就。 |

#### 6. 代价

管理者情绪和时间被大量消耗。

#### 7. 关键变量

- 容错度。
- 对员工发展和收入的理解。
- 保护服务对象的底线。

#### 8. 可参考人群

管理年轻团队、创意团队或高流动团队的人。

### 场景 3：AI 冲击模特行业时，要不要自己先下场做自媒体

> **一句话决策：** 她在 2023 年左右退出一部分公司经营，自己做自媒体，判断下一口饭在哪里。
> **核心变量：** AI 可能替代部分模特工作；如果自己不懂自媒体，就很难带模特转向新机会。
> **主要代价：** 从老板/经纪人变成内容创作者，需要重新学习平台规则并承受失败。

#### 1. 当时约束

- AI 和换脸等技术让模特工作存在被替代风险。
- 她心疼员工和模特，希望找到下一种活法。

#### 2. 备选项

| 选项 | 内容 |
| --- | --- |
| A | 继续守住原模特经纪业务。 |
| B | 只口头要求模特转型。 |
| C | 自己先做自媒体，跑通后再判断能否复制。 |

#### 3. 最终选择

共生入局，自己下场做网红/内容。

#### 4. 行动路径

- 用自己的账号测试自媒体路径。
- 如果跑得通，再思考如何带一部分符合特质的模特吃螃蟹；如果跑不通，就换方向。

#### 5. 结果

| 阶段 | 结果 |
| --- | --- |
| 短期 | 她把行业焦虑变成具体实验。 |
| 中长期 | 为模特经纪公司寻找从“被拍摄的人”到“有表达的人”的新可能。 |

#### 6. 代价

需要接受自己也可能跑不通。

#### 7. 关键变量

- 技术替代压力。
- 老板是否先自己试。
- 能否把试验结果转成组织方法。

#### 8. 可参考人群

传统行业被 AI 或平台变化冲击、需要亲自验证新路径的人。

### 场景 4：面对过往选择，要不要一直后悔

> **一句话决策：** 她把“原谅自己”作为自我关系的第一课：不要一直假设如果当初选 A 会更好。
> **核心变量：** 创业、婚恋、工作和形象选择都会产生后悔，但人生没有可验证的平行路径。
> **主要代价：** 需要放下对过去更优解的幻想。

#### 1. 当时约束

- 人会不断回想如果当初没有结婚、离婚、出轨、换工作会怎样。
- 模特也会后悔剪短发等具体选择。

#### 2. 备选项

| 选项 | 内容 |
| --- | --- |
| A | 持续沉溺如果当初。 |
| B | 否认错误。 |
| C | 承认已经选择了这条路，把当下过好。 |

#### 3. 最终选择

原谅自己，允许一切已经发生的选择。

#### 4. 行动路径

- 把注意力从假设过去移回当下行动。
- 鼓励模特也不要为已经发生的选择持续消耗。

#### 5. 结果

| 阶段 | 结果 |
| --- | --- |
| 短期 | 减少关系和自我攻击的内耗。 |
| 中长期 | 让她能继续创业、管理和转型。 |

#### 6. 代价

原谅不是一次完成，需要反复练习。

#### 7. 关键变量

- 能否停止反事实消耗。
- 能否把错误转成下一步行动。
- 自我关系是否稳定。

#### 8. 可参考人群

在重大选择后长期后悔、自责的人。
