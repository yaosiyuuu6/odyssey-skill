郑执的故事是一连串错过风口后的自我转向：19岁出道，没赶上青春文学盛世；后来做影视改编，发现话语权在导演手里；到《森中有林》时，他在半推半就中自己执导。

## 二、核心决策场景拆解

### 场景 1：19岁出道后，要不要继续吃写作这碗饭

> **一句话决策：** 前几本书让他觉得能写，但进入社会后写作很难养活自己。
> **核心变量：** 少年作家/青春文学窗口已经过去，出版收入不稳定。
> **主要代价：** 坚持写作可能长期贫困和不确定。

#### 1. 当时约束

- 他19岁写出处女作。
- 访谈中提到前两三本书后，进入社会发现养不了自己。

#### 2. 备选项

| 选项 | 内容 |
| --- | --- |
| A | 放弃写作。 |
| B | 转向当时更流行的包装路线。 |
| C | 继续写，同时寻找影视改编机会。 |

#### 3. 最终选择

继续写作，并逐渐进入影视改编。

#### 4. 行动路径

- 早期出版建立作者身份。
- 电影行业资金变多时，开始有人买小说版权。

#### 5. 结果

| 阶段 | 结果 |
| --- | --- |
| 短期 | 写作身份延续但收入不稳。 |
| 中长期 | 文学作品成为影视创作入口。 |

#### 6. 代价

坚持写作可能长期贫困和不确定。

#### 7. 关键变量

- 风口错位。
- 作者身份。
- 版权机会。

#### 8. 可参考人群

没有赶上行业红利、仍想继续创作的人。

### 场景 2：版权改编时，要不要只做编剧

> **一句话决策：** 他一开始参与改编自己的作品，但逐渐意识到编剧和原作者不掌握最终话语权。
> **核心变量：** 小说家有强表达欲，影视系统里导演拥有更多决定权。
> **主要代价：** 只做编剧容易被项目消耗，也会不断被绑定。

#### 1. 当时约束

- 访谈中提到买小说常要求他打包写剧本。
- 他不算职业编剧，基本改编自己的东西。

#### 2. 备选项

| 选项 | 内容 |
| --- | --- |
| A | 把版权卖掉后完全不管。 |
| B | 继续打包做编剧。 |
| C | 在合适项目中争取更高创作控制。 |

#### 3. 最终选择

继续参与自己的作品改编，但逐渐走向导演位置。

#### 4. 行动路径

- 写完库存改编项目。
- 在影视项目中积累剧组经验。

#### 5. 结果

| 阶段 | 结果 |
| --- | --- |
| 短期 | 获得影视行业入口。 |
| 中长期 | 对导演话语权有更强意识。 |

#### 6. 代价

只做编剧容易被项目消耗，也会不断被绑定。

#### 7. 关键变量

- 原作者表达欲。
- 影视工业分工。
- 话语权分配。

#### 8. 可参考人群

从文字、IP 或内容源头进入影视/产品制作的人。

### 场景 3：《森中有林》要不要自己导

> **一句话决策：** 在反复否定导演人选后，他半推半就自己上，完成第一次导演。
> **核心变量：** 他没有正式导演经验，但长期跟组和编剧经验提供基础。
> **主要代价：** 首导项目规模不小，失败会影响已成名作者身份。

#### 1. 当时约束

- 访谈中说老板和团队讨论导演人选，他提一个否一个。
- 他承认自己一开始没做好准备。

#### 2. 备选项

| 选项 | 内容 |
| --- | --- |
| A | 交给其他导演。 |
| B | 继续只做编剧。 |
| C | 自己执导。 |

#### 3. 最终选择

自己执导《森中有林》。

#### 4. 行动路径

- 基于同名小说和剧本进入导演位置。
- 用多年剧组经验支撑首导。

#### 5. 结果

| 阶段 | 结果 |
| --- | --- |
| 短期 | 承担新人导演压力。 |
| 中长期 | 从小说家/编剧扩展为导演。 |

#### 6. 代价

首导项目规模不小，失败会影响已成名作者身份。

#### 7. 关键变量

- 控制权。
- 经验迁移。
- 首作风险。

#### 8. 可参考人群

想从幕后文本创作者转向导演/主理人的人。
