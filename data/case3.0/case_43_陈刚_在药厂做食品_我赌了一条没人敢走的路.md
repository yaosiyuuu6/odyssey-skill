陈刚的决策链很清楚：创业初期靠诚信度过账上只剩两万的现金危机；产品选择上不追原创新药，而是在仿制药里找无人重视的品种；集采之后又转向被药厂鄙视的医学营养食品。

## 二、核心决策场景拆解

### 场景 1：账上只剩两万时，要不要卖掉公司

> **一句话决策：** 他没有卖公司，而是向政府寻求过桥资金，靠长期诚信获得信任。
> **核心变量：** 现金缺口约400万，公司销售差、开支大。
> **主要代价：** 借不到钱公司可能马上撑不住。

#### 1. 当时约束

- 2007年前后商业闭环未形成。
- 银行因政策原因不再贷款。

#### 2. 备选项

| 选项 | 内容 |
| --- | --- |
| A | 卖掉公司。 |
| B | 借高利贷。 |
| C | 向政府求助过桥资金。 |

#### 3. 最终选择

放弃高利贷，找政府过桥，保住公司。

#### 4. 行动路径

- 向县委书记打电话说明情况。
- 用政府资金过桥，完成银行切换。

#### 5. 结果

| 阶段 | 结果 |
| --- | --- |
| 短期 | 公司从现金危机中活下来。 |
| 中长期 | 他把“诚实”视为企业长期机会来源。 |

#### 6. 代价

借不到钱公司可能马上撑不住。

#### 7. 关键变量

- 诚信记录。
- 政府和银行信任。
- 现金流韧性。

#### 8. 可参考人群

创业公司遇到现金流断裂的人。

### 场景 2：没有实力做原创药时，要不要做冷门仿制药

> **一句话决策：** 他承认原创药需要巨大资金，于是选择仿制药里无人重视的品种，用文献和医生建立认知。
> **核心变量：** 资金和能力不足以支撑十年无销售的新药研发。
> **主要代价：** 仿制药不如原创药听起来高级。

#### 1. 当时约束

- 他学药，知道原创药需要极大资金。
- 第一个品种买下后销售很差。

#### 2. 备选项

| 选项 | 内容 |
| --- | --- |
| A | 硬做原创新药。 |
| B | 做热门仿制药拼价格。 |
| C | 找冷门品种，用文献和医生沟通做深。 |

#### 3. 最终选择

选择中枢神经系统等方向，用文献研究和医生讲课打开市场。

#### 4. 行动路径

- 去图书馆查文献。
- 公司人员到医院讲课，第一场由他自己讲。

#### 5. 结果

| 阶段 | 结果 |
| --- | --- |
| 短期 | 慢、笨，但逐步建立医生共同语言。 |
| 中长期 | 公司形成核心药品业务。 |

#### 6. 代价

仿制药不如原创药听起来高级。

#### 7. 关键变量

- 资金实力。
- 文献能力。
- 医生信任。

#### 8. 可参考人群

资源有限但想在专业行业做壁垒的人。

### 场景 3：集采冲击后，要不要转向医学营养食品

> **一句话决策：** 他从原本不相信食品，到布局医学营养食品，即使被说越做越低、研发人员离开。
> **核心变量：** 药品集采压缩价格，原有路径受冲击；食品在药厂内部被看低。
> **主要代价：** 团队出走和专业身份被质疑。

#### 1. 当时约束

- 访谈提到从2022年开始布局医学营养食品。
- 研发人员担心简历写食品以后找不到药厂工作。

#### 2. 备选项

| 选项 | 内容 |
| --- | --- |
| A | 继续只做药品。 |
| B | 卖掉公司退出。 |
| C | 进入医学营养食品。 |

#### 3. 最终选择

转向医学营养食品，把它作为公司三大品类之一。

#### 4. 行动路径

- 从国外研究中看到抗抑郁药效果不好时加食品的可能性。
- 用安全性、有效性等标准验证产品。

#### 5. 结果

| 阶段 | 结果 |
| --- | --- |
| 短期 | 被专家和员工质疑，团队有人离开。 |
| 中长期 | 打开药厂少有人走的新赛道。 |

#### 6. 代价

团队出走和专业身份被质疑。

#### 7. 关键变量

- 用户价值。
- 对偏见的修正。
- 长期投入。

#### 8. 可参考人群

主业受政策冲击、考虑跨界但被同行看低的人。
