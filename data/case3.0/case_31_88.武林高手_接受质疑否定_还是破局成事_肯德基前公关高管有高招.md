## 二、核心决策场景拆解

### 场景 1：加入项目组三个月，要不要接下全国三人篮球项目

> **一句话决策：** 虽然刚加入公司不久，她因为懂篮球被拉进项目组，选择承担一个没人做过、需要官方身份的全国项目。
> **核心变量：** 品牌要提前应对营养健康挑战；三人篮球门槛低、可全国复制；但官方背书和组织流程都不清晰。
> **主要代价：** 大量跨部门、跨城市、跨政府沟通，短期很难证明成果。

#### 1. 当时约束

- 2004年项目启动时，她加入公司约三个月。
- 部门里少有人懂篮球，项目还需要合法官方身份。
- 肯德基门店遍布全国，但要动员餐厅经理和员工。

#### 2. 备选项

| 选项 | 内容 |
| --- | --- |
| A | 做常规品牌活动。 |
| B | 把赛事外包给公关公司。 |
| C | 内部承担，从官方身份和门店组织开始搭建。 |

#### 3. 最终选择

成为核心成员之一，推动肯德基三人篮球项目。

#### 4. 行动路径

- 去北京找体育相关部门和协会，反复拜访争取支持。
- 把三人篮球设计为一小块场地、一个篮筐就能开展的普及赛事。
- 依托全国门店和餐厅团队组织落地。

#### 5. 结果

| 阶段 | 结果 |
| --- | --- |
| 短期 | 项目第一年覆盖大量城市，规模超出最初想象。 |
| 中长期 | 项目持续二十多年，十几年后仍能被参与者记起并形成品牌情感。 |

#### 6. 代价

长期耐得住寂寞，不能只看当下曝光。

#### 7. 关键变量

- 官方合法性。
- 门店网络动员能力。
- 项目是否能在五年、十年后还对品牌有作用。

#### 8. 可参考人群

需要把一个品牌活动做成长期组织资产的市场、公关或运营负责人。

### 场景 2：公关项目要不要追求立刻热闹

> **一句话决策：** 她选择把公关理解为品牌温度和长期信任，而不是用钱砸知名度。
> **核心变量：** 公关项目的效果常常延迟兑现；品牌喜欢需要长期积累。
> **主要代价：** 短期难以用即时指标证明价值。

#### 1. 当时约束

- 外部方案容易把知名度当目标。
- 真正的人心和信任不能被交易。

#### 2. 备选项

| 选项 | 内容 |
| --- | --- |
| A | 用预算换声量。 |
| B | 只做短期促销。 |
| C | 做能在多年后仍被记住的公共项目。 |

#### 3. 最终选择

坚持长期主义公关，把项目当成品牌房子的长期装修。

#### 4. 行动路径

- 要求团队思考五年、十年后项目对品牌的作用。
- 用赛事、青年沟通等方式让品牌进入真实生活场景。

#### 5. 结果

| 阶段 | 结果 |
| --- | --- |
| 短期 | 项目推进慢、协同难。 |
| 中长期 | 多年后仍有人因为三人篮球项目产生共同记忆。 |

#### 6. 代价

需要忍受看不见立刻回报的阶段。

#### 7. 关键变量

- 品牌活动是否真进入人的生活。
- 是否愿意等待长期反馈。
- 项目是否有公共性。

#### 8. 可参考人群

被短期声量指标裹挟的公关和品牌从业者。

### 场景 3：面对90后标签化，要不要用平等对话替代说教

> **一句话决策：** 她把“90后是吃着肯德基长大的一代”转成品牌沟通机会，选择和年轻人平等对话，而不是教育他们。
> **核心变量：** 年轻一代被批判、被贴标签；品牌若像家长或老师就会失去信任。
> **主要代价：** 要放弃控制感，让年轻人的真实需求进入项目。

#### 1. 当时约束

- 当时社会对90后有很多批判。
- 品牌需要理解而不是训导新一代消费者。

#### 2. 备选项

| 选项 | 内容 |
| --- | --- |
| A | 跟随社会批判年轻人。 |
| B | 做正式、居高临下的青年项目。 |
| C | 把品牌定位成朋友，支持年轻人表达和实现愿望。 |

#### 3. 最终选择

策划“肯德基对话90后”一类平等沟通项目。

#### 4. 行动路径

- 请媒体追踪90后画像，而不是只写肯德基。
- 把90后想做的事列清单，并帮助他们与家长、老师等对象对话。

#### 5. 结果

| 阶段 | 结果 |
| --- | --- |
| 短期 | 品牌获得更贴近年轻人的沟通方式。 |
| 中长期 | 形成一套用共情而非权威管理公众关系的方法。 |

#### 6. 代价

需要接受年轻人表达的不确定性。

#### 7. 关键变量

- 品牌是否愿意放低姿态。
- 是否真正研究人群，而不是使用标签。
- 是否能把消费者当成平等对象。

#### 8. 可参考人群

需要和新一代用户、员工或学生建立信任的人。
