## 二、核心决策场景拆解

### 场景 1：看到翼装飞行后，要不要进入一项高风险小众运动

> **一句话决策：** 2012 年在张家界看到翼装飞行比赛后，他选择学习翼装飞行，而不是停留在旁观。
> **核心变量：** 翼装飞行速度极高、早期事故率高，但也提供了近似飞鸟的身体体验。
> **主要代价：** 训练门槛高，低空翼装对失误几乎没有容错。

#### 1. 当时约束

- 翼装飞行进入中国较晚。
- 他最初也觉得这项运动危险：没有动力，只是穿着飞行服从山上跳下去。

#### 2. 备选项

| 选项 | 内容 |
| --- | --- |
| A | 只作为观众观看。 |
| B | 做普通跳伞。 |
| C | 系统学习跳伞、高空翼装、低空跳伞和低空翼装。 |

#### 3. 最终选择

从基础跳伞开始，系统进入翼装飞行。

#### 4. 行动路径

- 先完成数百次跳伞基础，再学习高空翼装、低空跳伞和低空翼装。
- 2012 年开始学习，持续十多年。

#### 5. 结果

| 阶段 | 结果 |
| --- | --- |
| 短期 | 进入一条训练密度和风险都很高的运动路径。 |
| 中长期 | 成为亚洲翼装飞行代表性人物之一。 |

#### 6. 代价

时间、身体和生命风险都高于普通运动。

#### 7. 关键变量

- 是否愿意按训练阶梯来。
- 是否理解风险而不是盲目兴奋。
- 长期训练投入。

#### 8. 可参考人群

被高风险兴趣吸引、需要判断是否系统进入的人。

### 场景 2：经历危险后，要不要放慢并重新敬畏风险

> **一句话决策：** 美国训练中开伞冲击导致颈部受伤、瑞士低空飞行接近极限高度后，他把危险当成警钟，要求自己更严谨。
> **核心变量：** 翼装不太允许犯错；进步快会带来自信，也会放大风险。
> **主要代价：** 每一次失误都可能没有修正时间。

#### 1. 当时约束

- 低空飞行时离山体、树和障碍物很近。
- 每秒下降高度高，开伞高度窗口很窄。

#### 2. 备选项

| 选项 | 内容 |
| --- | --- |
| A | 把危险当作刺激继续加速。 |
| B | 因危险退出。 |
| C | 保留热爱，但用更严谨科学的态度训练。 |

#### 3. 最终选择

继续飞行，但更认真地对待规则、装备和训练。

#### 4. 行动路径

- 保持每年大量实际飞行训练。
- 飞行前整理装备、检查降落伞、头盔、相机、高度表等每个环节。

#### 5. 结果

| 阶段 | 结果 |
| --- | --- |
| 短期 | 危险经历转化为风险意识。 |
| 中长期 | 形成“勇敢不是莽撞”的运动方法。 |

#### 6. 代价

必须接受长期处在风险边缘，同时不能被刺激感支配。

#### 7. 关键变量

- 风险复盘。
- 装备和流程纪律。
- 自信与敬畏的平衡。

#### 8. 可参考人群

从事高风险运动、创业或职业的人，需要区分勇敢和莽撞。

### 场景 3：要不要把飞行记录下来并与品牌合作

> **一句话决策：** 他选择用 Insta360 全景相机记录翼装飞行，并成为影石品牌大使，把个人运动经验转成公共影像。
> **核心变量：** 极限运动本身难被旁观者理解，影像能把体验和风险传递出去。
> **主要代价：** 记录设备必须轻、稳、安全，不能干扰飞行。

#### 1. 当时约束

- 国内早期翼装运动员少，也没有太多有意识记录。
- 普通相机角度有限、负重较大。

#### 2. 备选项

| 选项 | 内容 |
| --- | --- |
| A | 只飞行，不记录。 |
| B | 用普通相机记录。 |
| C | 使用全景相机，和影石合作呈现飞行视角。 |

#### 3. 最终选择

用全景相机记录训练和飞行，并与影石合作。

#### 4. 行动路径

- 2020 年前后开始在社交平台发布飞行视频。
- 通过头盔、胸前、脚部等机位记录不同视角，且符合安全规则。

#### 5. 结果

| 阶段 | 结果 |
| --- | --- |
| 短期 | 更多人能看见翼装飞行的真实视角。 |
| 中长期 | 他的个人形象与影石 Think bold 的品牌精神结合。 |

#### 6. 代价

要在创作欲和安全要求之间保持边界。

#### 7. 关键变量

- 记录是否影响安全。
- 品牌价值是否与个人实践匹配。
- 影像能否帮助理解运动而非制造猎奇。

#### 8. 可参考人群

希望把小众专业或高风险经验转成内容表达的人。
